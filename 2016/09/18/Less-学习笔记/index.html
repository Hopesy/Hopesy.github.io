Less 学习笔记

  • Less


前言

Less 是一门 CSS 预处理语言，它扩展了 CSS 语言，增加了变量、Mixin、函数等特性，使 CSS 更易维护和扩展。
Less 可以运行在 Node 或浏览器端。

笔记

注释


1./我是被编译的/
2.//我不会被编译

变量


1.@test_width:300px;
混合

.box{
width:@test_width;
height:@test_width;
background-color: yellow;

.border;
}
.border
{
border:solid 5px pink;
}
.box2{
.box;
margin-left:100px;
}


混合-可带参数

.border_test(@border_width){
border:solid yellow @border_width;
}
.test_hunhe{
.border_test(30px);
}

混合-带默认值

.border_test(@border_width:30px){
border:solid yellow @border_width;
}
.test_hunhe1{
.border_test();
}
.test_hunhe2{
.border_test(20px);
}

匹配模式



.triangle(top,@w:5px,@c:#ccc){
border-width:@w;
border-color: transparent transparent @c transparent;
border-style: dashed dashed solid dashed;
}
.triangle(bottom,@w:5px,@c:#ccc){
border-width:@w;
border-color:@c transparent transparent transparent;
border-style: solid dashed dashed dashed;
}
//@为默认格式，而且后面的声明不能省略
//无论匹配到哪个，一下默认内容均会生成
.triangle(@,@w:5px,@c:#ccc){
width: 0;
height: 0;
overflow:hidden;
}
//如果没有匹配，只会生成默认内容
.sanjiao1{
.triangle(top,80px)
}

运算

任何数字、颜色或者变量都可以参与运算，运算应该被包裹在括号中

@test_01:300px;
.box_01{
width:(@test_01 - 20) * 5;
color:#ccc - 10;
}

嵌套规则


.list{
width:600px;
margin:30px auto;
padding:0;
list-style: none;
li {
height: 30px;
line-height:30px;
background-color: pink;
maigin-bottom:5px;
}
a{
float:lest;
//代表上一层选择器，此处指a
&：hover{red}
}
}

@arguments变量


.border_arg(@w:30px,@c:red,@xx:solid){
//@arguments会传入所有声明的变量
border:@arguments;
}
.test_arguments{
.border_arg(40px);
}

避免编译

.test_03{
//~’’里面的内容不会被less编译
width: ~’calc(300px - 30px)’;
}

结语
